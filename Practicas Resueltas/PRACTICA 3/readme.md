# Importante: 
Un error que fui arrastrando en los ejercicios es que,**TODA FUNCION DEBE DEVOLVER UN VALOR**  
( USAR RETURN EN FUNCIONES. NO DEBEMOS USAR EXIT. )  

RETURN: RETORNA UN CODIGO ENTRE 0 Y 255  
EXIT: CORTA LA EJECUCION DEL PROGRAMA.  

TANTO PARA RETURN COMO PARA EXIT SE PUEDEN USAR CODIGOS DEL 0 AL 255.  

EL CODIGO 0 INDICA CASO EXITOSO  
CODIGOS DEL 1 AL 255 INDICAN CASO FALLIDO
